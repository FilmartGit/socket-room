# Socket Room - Онлайн чат на React и Node.js

Socket Room — это веб-приложение для общения в реальном времени, реализованное по принципу "одна комната — один чат". Проект состоит из двух частей: фронтенд на React.js и бэкенд на Node.js с использованием Express и Socket.IO.

## Особенности

- **Реальное время**: Общение происходит мгновенно благодаря WebSocket через Socket.IO.
- **Простота использования**: Пользователь вводит имя и выбирает комнату — и сразу может общаться.
- **Эмодзи-пикер**: Удобный выбор эмодзи прямо в интерфейсе чата.
- **Простая навигация**: Маршрутизация на стороне клиента с помощью `react-router-dom`.

## Технологии

### Бэкенд (`/server`)
- **Node.js** — серверная платформа
- **Express.js** — веб-фреймворк
- **Socket.IO** — двусторонняя связь в реальном времени
- **CORS** — разрешение кросс-доменных запросов
- **Nodemon** — перезагрузка сервера при изменении кода

### Фронтенд (`/client/app`)
- **React.js** — библиотека для построения интерфейсов
- **React Router DOM** — клиентская маршрутизация
- **Socket.IO Client** — подключение к серверу в реальном времени
- **Emoji Picker React** — выбор эмодзи
- **React Icons** — иконки в интерфейсе
- **CSS Modules** — стилизация компонентов

## Структура проекта

```
socket-room/
├── client/
│   └── app/               # React-приложение
│       ├── public/
│       └── src/
│           ├── app/       # Компоненты приложения (Main, Chat)
│           ├── routes/    # Маршруты
│           ├── ui/        # UI-компоненты (кнопки, инпуты)
│           └── styles/    # Стили и шрифты

    ├── services/        # Логика обработки событий Socket.IO
    ├── route/           # Роуты (если используются)
    ├── constants/       # Константы (например, список комнат)
    └── index.js         # Точка входа сервера
```

## Запуск проекта

### 1. Запуск сервера

```bash
cd server
npm install
npm run dev
```

Сервер будет доступен на `http://localhost:5000`.

### 2. Запуск клиента

```bash
cd client/app
npm install
npm start
```

Фронтенд откроется в браузере по адресу `http://localhost:3000`.
